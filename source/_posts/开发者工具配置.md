## pip3 配置清华镜像源加速

```bash
pip3 install -U pip
pip3 config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

## virtualenvwrapper 配置

```bash
pip3 install --user virtualenv
pip3 install --user virtualenvwrapper
which virtualenvwrapper.sh
# /Users/evan/Library/Python/3.8/bin/virtualenvwrapper.sh
which python3
# /usr/bin/python3

echo -e '\n\n## virtualenvwrpper config \nexport WORKON_HOME="~/.envs" \nexport VIRTUALENVWRAPPER_PYTHON="/usr/bin/python3" \nsource /Users/evan/Library/Python/3.8/bin/virtualenvwrapper.sh' >> ~/.zshrc

source ~/.zshrc
```

## npm 配置阿里巴巴镜像加速

```bash
npm config get registry
# https://registry.npmjs.org/ (默认官方地址）
npm config set registry http://registry.npmmirror.com
# http://registry.npmmirror.com 同步频率快，稳定，速度快
```

## yarn 配置阿里巴巴镜像加速

```bash
npm install --location=global yarn
yarn config get registry
# https://registry.yarnpkg.com
yarn config set registry http://registry.npmmirror.com
# 或者
yarn config set registry https://npm.taobao.org
```

## docker 配置镜像加速

```bash
# 针对 Ubuntu 和 Centos 进行配置
vim /etc/docker/daemon.json

{
	"registry-mirrors":[
			"https://mirror.ccs.tencentyun.com",
			"https://docker.mirrors.ustc.edu.cn/",
			"https://yiitcyjr.mirror.aliyuncs.com"
	]
}

sudo systemctl daemon-reload
sudo systemctl restart docker
```

## yum 清华镜像源加速

```bash
# 对于 CentOS 7
sudo sed -e 's|^mirrorlist=|#mirrorlist=|g' \
         -e 's|^#baseurl=http://mirror.centos.org|baseurl=https://mirrors.tuna.tsinghua.edu.cn|g' \
         -i.bak \
         /etc/yum.repos.d/CentOS-*.repo

# 对于 CentOS 8
sudo sed -e 's|^mirrorlist=|#mirrorlist=|g' \
         -e 's|^#baseurl=http://mirror.centos.org/$contentdir|baseurl=https://mirrors.tuna.tsinghua.edu.cn/centos|g' \
         -i.bak \
         /etc/yum.repos.d/CentOS-*.repo

sudo yum makecache
```
